## [0.1.0] - 2026-04-07

### Added

- Initial JupyterLab 4.x prebuilt extension scaffold.
- Hello world plugin (`jupyterlab-datatable-renderer:plugin`) that logs a
  message on activation.
- Python package `jupyterlab_datatable_renderer` with hatchling +
  hatch-jupyter-builder build configuration.
- GitHub Actions workflows for build, isolated install test, and release.
